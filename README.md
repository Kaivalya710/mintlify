This project is a desktop-only front-end recreation of the Mintlify landing page.

The objective was to closely replicate the original websiteâ€™s layout, typography, colors, spacing, and visual structure.

---

## Sections Recreated

The following major sections of the Mintlify landing page were recreated:

- Navigation Bar with logo, links, and CTA buttons  
- Hero Section with badge, heading, description, input field, and preview image  
- Trusted Companies / Logos section  
- Intelligence introduction section  
- Feature Cards section highlighting key platform capabilities  
- AI Assistant section  
- Enterprise section with feature grid  
- Customer Story / Highlight section  
- Customer showcase / industry cards section  
- Final Call-to-Action section  
- Footer with multi-column links, social icons, and status indicator  

---

## Fonts Used

Primary font:

- Inter (Google Fonts)

Fallback stack:

- system UI fonts (Segoe UI, Roboto, Ubuntu, etc.)

The typography was selected to closely match the original Mintlify website.

## Assets

- Images and icons were chosen to resemble the original Mintlify website as closely as possible  
- SVG icons were used where available to maintain visual accuracy  
- All assets are static and locally referenced  

---

## Tech Stack

- HTML5  
- CSS3 (Flexbox and Grid for layout)  
- Google Fonts  
- Static image and icon assets  

---

## Notes

- This project focuses on visual recreation rather than functionality  
- Interactive features from the original website were intentionally omitted due to constraints  
- The layout aims to be visually accurate on desktop screens
